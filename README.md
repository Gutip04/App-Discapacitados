###  Documentación Completa - Sistema App Discapacitacion
     Resumen 
        App Discapacitacion es un sistema integral para la gestión y análisis de información de pacientes con discapacidad. Desarrollado con arquitectura moderna full-stack, proporciona herramientas completas para registro, seguimiento y generación de reportes estadísticos.

Arquitectura del Sistema
    Frontend (React + TypeScript)
        Framework: React 19 con TypeScript
        Build Tool: Vite para desarrollo rápido
        Estilos: Tailwind CSS + Lucide React
        Estado: Context API + Custom Hooks
        Ruteo: React Router DOM
        HTTP Client: Axios
    Backend (Node.js + Express + TypeScript)
        Runtime: Node.js con Express
    Base de Datos: MySQL con MySQL2
        Autenticación: JWT + bcrypt
        Validación: Zod schemas
    Arquitectura: MVC 

Base de Datos (MySQL)
    Esquema: Relacional normalizado
    Vistas: Especializadas para reportes
    Scripts: Migraciones y datos base automatizados
